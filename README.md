# Savory Spot 

Savory Spot is a mobile application that helps users find restaurants based on budget, cuisine, rating, and location.

## Features
 Restaurant search,
 Filters (price, rating, cuisine),
 Favorites,
 Map navigation,
 User profile.

## Technologies Used
 React Native,
 Firebase (Authentication & Database),
 Google Maps API.
